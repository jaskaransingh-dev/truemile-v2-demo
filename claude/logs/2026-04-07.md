## Decisions made

- **MCI fallback: outbound_mci=0 = no data.** Seed rows with mci=0 are treated as "no real data" and trigger nearest-city proxy. This is correct because the seed SQL defaults all cities to 0.
- **Demo state via localStorage.** Rate con processing state shared between list and detail pages via `localStorage` key `truemile_demo_ratecon`. Simple, no context/store needed for demo.
- **TrailerType source of truth: chrome.storage.local.** After failed attempts at DAT DOM detection (timing races, Angular rendering), reverted to popup dropdown → storage → content script read. DRY_VAN is the default.
- **No client-side TRAILER_MISMATCH filter.** Removed in the revert session. Backend decision-engine-v2 handles trailer mismatch via FLEX/exact-match logic.
